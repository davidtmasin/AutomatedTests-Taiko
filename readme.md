### Como instalar o Taiko?

` npm install -g taiko`

---

### Executar um script taiko

- Para executar o script com visão apenas do CLI

`taiko <script_de_teste>.js`

**ou**

- Para executar o script com visão do navegador

` taiko <script_de_teste>.js --observe`
